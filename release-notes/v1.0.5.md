# MasterLLM v1.0.5 Pre-Release

## What's New

### Improved Browse UI
- **Compact Controls**: Device info and filters now shown as compact chips instead of large cards
- **Device Info Button**: Shows RAM and device profile - tap to see full details
- **Filters Button**: Shows active filter count - tap to open filter panel
- **Sort Button**: Shows current sort - tap to open filter panel
- **More Screen Space**: Model list now takes up most of the screen for easier browsing

### Discovery Controls Dialog (Bottom Sheet)
- **Modal Bottom Sheet**: All filters now open in a slide-up panel
- **Sort by**: Trending, Recently Updated, Most Downloaded with icons
- **Task Type Filter**: Filter by model task (text-generation, text-to-image, etc.)
- **Format Filter**: GGUF, SafeTensors, Diffusers with visual chips
- **Parameter Size Filter**: <=3B, 3B-8B, 8B-20B, 20B+ ranges
- **Apply Button**: Search button at bottom to apply all filters at once

### Enhanced Download Progress UI
- Real-time download progress visualization with detailed telemetry
- Network speed indicator showing download speed in MB/s
- Complete file list showing all files in a download bundle
- Visual progress bars with percentage completion
- Current file indicator in multi-file downloads

### Device Compatibility Detail Screen
- Tap on device info chip to view detailed specs
- Shows Total RAM and Available RAM
- CPU core count detection
- GPU info (Adreno/Mali/Xclipse/PowerVR detection)
- Vulkan support status
- Recommended maximum model parameter size for your device
- Compatibility guide explaining badge colors

### Real-time Search with Debouncing
- Search now triggers automatically as you type
- 300ms debounce to prevent excessive API calls
- Live search indicator shows when search is active
- Results count displayed below search bar

### Enhanced Filters (Hugging Face Style)
- **Sort by**: Trending, Recently Updated, Most Downloaded
- **Task Type Filter**: Text generation, text-to-image, and more
- **Format Filter**: GGUF, SafeTensors, Diffusers, or Any
- **Parameter Size Filter**: <=3B, 3B-8B, 8B-20B, 20B+
- Visual filter chips with checkmarks for selected filters

### Pagination Improvements
- First/Last page navigation buttons
- Page counter (current/total)
- Results range display
- Disabled state for navigation buttons when at boundaries
- Navigation controls at both top and bottom of results

### Active Downloads Section
- Downloads tab now shows active downloads
- Real-time progress for each downloading model
- Visual file list for multi-file bundles
- Speed and percentage indicators

### Light Theme Implementation
- New Green Apple primary color scheme
- Cinnamon secondary color for warm accents
- Dark Coffee tertiary color for depth
- Creamy off-white background for comfortable reading
- Properly themed for light mode throughout the app

## Technical Improvements
- Improved device profile detection
- Better error handling in downloads
- Enhanced UI responsiveness
- Memory optimization for model compatibility checking

## APK Details
- **Debug APK**: ~80MB (includes debugging symbols)
- **Release APK**: ~26MB (optimized with ProGuard)
- **Min SDK**: 26 (Android 8.0)
- **Target SDK**: 35 (Android 15)
- **Version**: 1.0.5 (versionCode 5)

## Installation
Download the appropriate APK for your needs:
- Use `MasterLLM-v1.0.5-debug.apk` for testing and development
- Use `MasterLLM-v1.0.5-release.apk` for daily use (smaller, optimized)

## Compatibility
- Android 8.0 (API 26) and above
- ARM64 and x86_64 architectures
- Vulkan-compatible devices recommended for GPU acceleration
